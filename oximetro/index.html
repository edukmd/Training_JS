Sistema de Monitoramento UFSCar - Prova de Conceito

Sobre o Projeto

Este é um projeto aberto para auxiliar no monitoramento remoto, e sem fio (Wifi), de pacientes através da medida de batimentos cardíacos, oxigenação do sangue e temperatura. O sistema coleta dados automaticamente em envia dados para um servidor, em nuvem, que possibilita acompanhamento remoto dos pacientes.

O dispositivo, pode ser construído, com custo de protótipo por cerca de R$ 75,00 usando uma placa ESP8266 (Wemos D1 Mini), um sensor MAX30100, bateria, fios e peças fabricadas por impressora 3D.
